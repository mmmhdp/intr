### Реализация LRU-кэша на C ([caching](https://github.com/mmmhdp/bscalg/tree/main/calg/caching))

*Личный / Образовательный проект*

Реализован **кэш с алгоритмом наименее недавно используемого (LRU)** на чистом C с поддержкой операций за константное время

Построены ключевые структуры данных: **хеш-таблица** для быстрого поиска ключ → узел и **двусвязный список** для отслеживания порядка использования (от наиболее недавно использованного до наименее) с обеспечением **O(1) операций get/put**

Разработана логика вытеснения, автоматически удаляющая наименее недавно использованную запись при переполнении кэша

Выполнено ручное управление памятью, включая динамическое выделение, связывание указателей и корректное освобождение памяти

Разработаны комплексные модульные тесты, проверяющие корректность работы кэша, политику вытеснения и обработку граничных случаев

Структурирован код для модульности и повторного использования, с чёткими интерфейсами между структурами данных и логикой кэша

**Технологии:** C, Хеш-таблица, Двусвязный список, Управление памятью, Модульное тестирование, Git
